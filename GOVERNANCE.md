# Governance

## Project Leadership

**Maintainer:** [@cannatoshi](https://github.com/cannatoshi)

## Roles

| Role | Description | Permissions |
|------|-------------|-------------|
| **Maintainer** | Project owner, final decisions | Full access |
| **Committer** | Trusted contributors | Merge PRs, manage issues |
| **Contributor** | Anyone who contributes | Submit PRs, open issues |

## Decision Making

1. Discussion in GitHub Issues or Discussions
2. Consensus among active contributors
3. Final authority: Maintainer

## Becoming a Committer

- Sustained quality contributions over 3+ months
- Nominated by maintainer
- Demonstrated understanding of project goals
